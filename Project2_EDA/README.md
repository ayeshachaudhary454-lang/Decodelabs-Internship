# Project 2: Exploratory Data Analysis (EDA) 📊

**DecodeLabs Industrial Training Kit — Batch 2026**
**Prepared by:** Ayesha Sarwar

## Goal
Analyze the cleaned e-commerce dataset to uncover patterns, trends, distributions, and outliers.

## Files
| File | Description |
|---|---|
| `Project2_EDA_AyeshaSarwar.ipynb` | Jupyter notebook with full EDA |
| `cleaned_dataset.csv` | Input dataset (from Project 1) |
| `eda_summary.csv` | Exported descriptive statistics |

## Steps Covered
1. Descriptive statistics (mean, median, std, min, max)
2. Univariate analysis (TotalPrice, UnitPrice, Quantity)
3. Categorical analysis (OrderStatus, PaymentMethod, Product)
4. Monthly sales trend
5. Outlier detection using IQR method
6. Correlation analysis
7. Business insights and "So What?" summary

## Key Findings
- Laptop generated the highest total revenue
- Delivered is the most common order status
- IQR outlier detection flagged high-value orders for investigation
- Quantity and TotalPrice show strong positive correlation

## Tech Stack
- Python 3.11, pandas, NumPy, Jupyter Notebook
