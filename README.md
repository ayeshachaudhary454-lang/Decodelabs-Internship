## About

This repository contains my project work completed during the DecodeLabs Data
Analytics internship. Each project builds on the last — starting with raw, messy
data and ending with polished, decision-ready visual insights — mirroring the
real workflow of a Data Analyst: clean the data, explore it, query it, then
communicate it.

## Projects

| # | Project | Folder | Focus | Key Skills |
|---|---------|--------|-------|------------|
| 1 | Data Cleaning | [`Project1_DataCleaning`](./Project1_DataCleaning) | Handling missing values, fixing inconsistent formatting, preparing raw data for analysis | Data cleaning, pandas, data validation |
| 2 | Exploratory Data Analysis (EDA) | [`Project2_EDA`](./Project2_EDA) | Uncovering patterns, trends, distributions, and outliers in the cleaned dataset | EDA, descriptive statistics, pandas |
| 3 | SQL | [`Project3_SQL`](./Project3_SQL) | Querying and analyzing the dataset using SQL | SQL, querying, aggregation |
| 4 | Data Visualization | [`Project4_DataVisualization`](./Project4_DataVisualization) | Turning analysis into clear, boardroom-ready charts and executive insights | Data visualization, chart selection, storytelling with data |

Each project folder contains its own `README.md` with full details, a Jupyter
notebook, and the dataset(s) used.

## The Journey

1. **Project 1 — Data Cleaning:** Started with raw e-commerce order data and
   fixed missing values, inconsistent formats, and other quality issues.
2. **Project 2 — EDA:** Explored the cleaned dataset to surface patterns, trends,
   and outliers worth investigating further.
3. **Project 3 — SQL:** Queried the dataset directly to answer specific business
   questions using SQL.
4. **Project 4 — Data Visualization:** Converted findings into clear, honest,
   executive-ready charts — following an Architect → Editor → Storyteller
   framework (define the question, keep the chart honest, then tell the story).

## How to Run Any Project

1. Open the relevant project folder.
2. Make sure its dataset file(s) are in the same folder as its notebook.
3. Install dependencies (varies slightly by project, generally):
```bash
   pip install pandas numpy matplotlib
```
4. Open the notebook in Jupyter and run all cells:
```bash
   jupyter notebook <notebook_name>.ipynb
```
